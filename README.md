Nomadi is a project that deals with advanced search of hotels around the world, offers the availability to see comments and pictures of the facilities, also in your personal area you can find your trips with a little planning so you do not miss anything of the city you are visiting

## Execution examples

    First step:
        - Run XAMPP
            - Run Apache
            - Run MySQL
    Second step:
        - Open browser
    Third step:
        - Search for localhost
    Fourth step:
        - Select the folder Nomadi
